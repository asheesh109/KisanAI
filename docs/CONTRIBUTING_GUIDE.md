**Contributing Guide (short)**

1. Read the Code of Conduct.
2. Pick an issue from `good first issue` or `ISSUE_BANK.md`.
3. Comment on the issue to claim it.
4. Fork the repo, create a branch, open a PR with the template filled.
5. Tests and type-check should pass; maintainers will review.

Where to get help: open a discussion or question issue.
