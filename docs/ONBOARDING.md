# Onboarding Guide for New Contributors

Welcome! This guide helps first-time contributors get started quickly and confidently.

1. Fork the repository and clone your fork.

```bash
git clone https://github.com/<your-username>/KisanAI.git
cd KisanAI
npm ci
cp .env.example .env.local
```

2. Create a feature branch:

```bash
git checkout -b feat/short-description
```

3. Run the app locally:

```bash
npm run dev
```

4. Find issues labeled `good first issue` or `beginner friendly` in Issues.

5. To claim an issue, comment on it and follow the steps in `docs/ISSUE_CLAIMING.md`.

6. Open a PR with tests and link the issue. Use the PR template.

Help and mentorship: add a comment to the issue asking for a mentor if you want guidance.
