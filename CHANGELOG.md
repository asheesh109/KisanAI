## Release Notes

### Version 0.1.0 (Beta)

**Release Date**: December 15, 2024

This is the initial beta release of KisanAI. The platform provides a foundation for AI-powered farmer assistance with core features implemented.

**Installation**: [See SETUP.md](./docs/SETUP.md)

**Known Issues**:
- Voice recognition accuracy varies by dialect
- Offline functionality limited in beta
- Database persistence not yet implemented
- Real-time data updates not active

**Next Version Highlights**:
- Improved voice recognition
- Enhanced crop disease analysis
- User authentication
- Real-time market data integration

## Contribution Guidelines

When contributing, please follow this changelog format for your PRs:

- Use semantic versioning (Major.Minor.Patch)
- Categorize changes: Added, Changed, Deprecated, Removed, Fixed, Security
- Link related issues
- Keep entries clear and concise

Example commit message:
```
feat(voice): improve hindi speech recognition

- Added noise filtering
- Updated language model
- Improved accuracy by 15%

Closes #123
```
